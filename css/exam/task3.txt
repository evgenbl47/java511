Задание 3. Реализовать страницу сайта-дневника с профилем пользователя. 
На этой странице студент может изменить информацию о себе или пароль для входа. Страница состоит из трех блоков: 
- загрузка изображения профиля; 
- форма с личными данными: имя; фамилия; почта; номер телефона; адрес; ссылка на социальную сеть.
- форма для изменения пароля. На этой странице активным должен быть пункт меню Settings. При выполнении задания обязательно использовать Bootstrap.
